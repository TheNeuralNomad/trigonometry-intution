## Why (cosθ, sinθ)?

Any point on the unit circle can be written as (x, y).

From the geometry:
- x = cosθ
- y = sinθ

So the point becomes (cosθ, sinθ).

## Key Angles and Coordinates

- 0° → (1, 0)  
- 90° → (0, 1)  
- 180° → (-1, 0)  
- 270° → (0, -1)  
- 360° → (1, 0)

## Trigonometric Values

| Angle | cosθ | sinθ | tanθ |
|------|------|------|------|
| 0°   | 1    | 0    | 0    |
| 90°  | 0    | 1    | undefined |
| 180° | -1   | 0    | 0    |
| 270° | 0    | -1   | undefined |
| 360° | 1    | 0    | 0    |

## Signs in Different Quadrants

- Quadrant I → sin +, cos +, tan +  
- Quadrant II → sin +, cos -, tan -  
- Quadrant III → sin -, cos -, tan +  
- Quadrant IV → sin -, cos +, tan -
